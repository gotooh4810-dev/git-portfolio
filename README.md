# git-portfolio

このプロジェクトは、WEB制作会社への転職活動用に作成したポートフォリオ作品です。

## 🚀 公開URL
https://gotooh4810-dev.github.io/git-portfolio/

## 🛠 使用技術
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- 制作ツール：VSCode / GitHub Desktop / Figma

## ✨ こだわったポイント
- **Vanilla JSへの移行:** jQueryに頼らず、ネイティブなJavaScriptでアコーディオンメニューやスライダーを実装しました。
- **レスポンシブ対応:** `clamp()` 関数を活用し、デバイスサイズに応じて文字サイズが滑らかに変化するフル流動タイポグラフィを採用しています。
- **Git運用:** 実務を意識し、作業単位ごとに細かくコミットメッセージを残すよう心がけました。
